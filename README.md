# Modern Personal Portfolio Website
A sleek, responsive, and interactive personal portfolio website built using HTML5, CSS3, and Vanilla JavaScript. This project features a modern dark-themed UI with smooth scrolling and custom animations.

# Features
Responsive Design: Fully optimized for desktops, tablets, and mobile devices using CSS Media Queries.

Sticky Navigation: A fixed header that remains accessible while scrolling.

Interactive UI: Integrated with Boxicons for professional social media and service icons.

Project Showcase: A dedicated portfolio section with image hover effects and overlay layers.

Smooth Scrolling: Seamless transitions between sections (Home, About, Services, Portfolio, Contact).

Contact Form: A styled user interface for inquiries and messages.

# Tech Stack
HTML5: Semantic structure for the web content.

CSS3: Custom styling, Flexbox, Grid layout, and transitions.

JavaScript: Logic for active navigation links and scroll-based effects.

Typography: Poppins font via Google Fonts.

Icons: Boxicons library.

# File Structure
Plaintext
├── portfolio.html   # Core HTML structure and scroll-logic JavaScript
├── style.css        # Custom CSS variables, animations, and responsive breakpoints


# Customization
To personalize this portfolio, you can modify the following:

Identity: Replace "Mariam Memon" in portfolio.html with your name and update the "About" text.

Branding: Change the --main-color (#0ef) in the :root section of style.css to your preferred accent color.

Media: Swap the placeholder image files (home.jpg, etc.) with your own professional photos or project screenshots.
